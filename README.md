## 📸 About the Project

C Vision is a security camera solutions company operating out of Alaba International Market, Lagos, with 15+ years of industry experience and 5,000+ cameras installed. This website was built to give them a professional online presence and help potential clients explore products and request quotes.

### What the site covers:
- Full product catalog (IP Dome, PTZ, NVR Systems, ColorVu cameras)
- Detailed Hikvision camera listings with specs and resolution info
- Services overview — consultation, installation, support, maintenance
- Branch locations with addresses and contact details
- Quote request / contact form
- Client testimonials section

## 🚀 Getting Started

### Prerequisites

Make sure you have these installed:
- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or yarn

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/cvision-website.git

# 2. Navigate into the project folder
cd cvision-website

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Then open your browser and go to: `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The output will be in the `/dist` folder, ready to deploy.

### Preview Production Build

```bash
npm run preview
```
